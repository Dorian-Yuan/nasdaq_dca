# NASDAQ 100 DCA Strategy Tool (纳指100定投评估工具)

这是一个基于 Python 和 GitHub Actions 构建的完全自动化的纳斯达克 100 (QQQ) 定投策略评估工具。它可以每天抓取金融数据，通过预设的指标判断当前市场处于什么位置，并给出“加倍定投”、“普通定投”或“暂停定投”的建议。

项目自带一个现代化的 PWA (渐进式 Web 应用) 界面，您可以将其**添加到手机主屏幕**，获得类似原生 App 的体验。

## ✨ 核心功能 (Model 4.3.2)

1. **双宽基指数支持**：同时支持纳斯达克 100 (^NDX) 和 标普 500 (^GSPC)，覆盖成长科技与蓝筹价值双主线。
2. **三维加权模型 (Asymmetric Accumulation)**：
   - 🏷️ **估值因子**：调用蛋卷 API，获取历史百分位。采用“泡沫不空仓，暴跌狂吸筹”的非对称逻辑。
   - 😱 **情绪因子**：接入 Yahoo Finance 获取期权隐含波动率 (VXN/VIX)。越恐慌得分越高。
   - 📉 **趋势因子**：计算与 200 日线乖离率。均线之上顺势增强，跌破防守，极度崩盘巨量加码。
3. **风格迁移加权 (Style-Tilted Weighting)**：根据两者的基本面基因，为其量身定制单项因子权重：
   - **纳指 (NDX)**：[估值 30% : 情绪 30% : 趋势 40%] -> 顺应科技股高动量，偏向**趋势跟踪**。
   - **标普 (SP500)**：[估值 40% : 情绪 30% : 趋势 30%] -> 顺应传统白马，偏向**价值均值回归**。
4. **全自动无服务器运行**：完全依赖 GitHub Actions 充当免费的定时服务器，每天美股收盘后自动运行。
5. **本地可视化沙盘 (GUI)**：提供纯本地的 `backtest_gui.html` 工具，拖动滑块即可瞬间推演出长达 4 年的收益率/超额 Alpha 图表，**代码与数据 100% 留存在您的电脑，绝不上传**。
6. **支持 Bark 推送**：自带配置参数，在每天北京时间 14:00 自动向您的手机 Bark 发送包含数据与决策的通知推送。

## 🚀 部署配置指南 (详细版)

这个项目设计为“Fork 即用”。为了保障您的隐私（如 GitHub Token 和 Bark Key 不被泄露），所有的敏感数据都是在本地配置文件和 GitHub Secrets 中独立管理的。

### 第 1 步：克隆/Fork 仓库并配置本地环境

1. 将本仓库 Fork 到您自己的 GitHub 账号下。
2. 将代码克隆到您的本地电脑。
3. 在本地项目根目录下找到 **`config.js`** 文件（如果不存在请新建，本仓库的 `.gitignore` 已经为您排除了它，防止上传泄露）。
4. 在 `config.js` 中填入如下代码，并替换为您自己的信息：
   ```javascript
   const CONFIG = {
       GITHUB_USERNAME: "您的GitHub用户名", 
       GITHUB_REPO_NAME: "包含当前代码的仓库名",
       GITHUB_TOKEN: "您的Personal Access Token (PAT)" // 可选：只有想在网页上手动点刷新按钮才需要配置
   };
   ```
5. *(注意：如果您需要配置 `GITHUB_TOKEN`，请去 GitHub 的 `Settings` -> `Developer settings` -> `Personal access tokens (classic)` 创建一个带有 `repo` (读写仓库)和 `workflow` 权限的 Token。)*

### 第 2 步：开启 GitHub Actions 读写权限 (关键❗)

如果不开启此项，每天 Actions 算好数据后将会因为没有权限，无法把结果写入到 `data.json` 中保存。

1. 进入您的 GitHub 仓库网页。
2. 点击顶部的 **Settings** 选项卡。
3. 在左侧菜单找到 **Actions**，点击展开后选择 **General**。
4. 一直往下滑，找到 **Workflow permissions** 模块。
5. 勾选 **Read and write permissions**，然后点击下方的 **Save** 按钮。

### 第 3 步：部署静态网页 (GitHub Pages)

利用 GitHub Pages 即可免费托管前端展示页面。

1. 同样在仓库的 **Settings** 页面。
2. 在左侧菜单，接近底部找到 **Pages**。
3. 在 `Build and deployment` -> `Source` 中，选择 **Deploy from a branch**。
4. 在下方的 `Branch` 下拉菜单中选择 **main**（或 master），文件夹保留 `/(root)`。
5. 点击 **Save**。
6. 等待约 1 分钟后刷新该页面，顶部就会显示您专属的网页链接了（例如 `https://您的名字.github.io/仓库名`）。

### 第 4 步：(可选) 配置每日 Bark 推送通知

如果您想每天下午 14:00 在手机上收到如下提示音响亮的推送通知，请配置您的 Bark 密钥。

1. 在仓库的 **Settings** -> 左侧边栏找到 **Secrets and variables** -> 展开选 **Actions**。
2. 点击绿色的 **New repository secret** 按钮。
3. **Name (名称)**：严格填入 `BARK_KEY` (全部大写，不要有空格)。
4. **Secret (数值)**：打开您手机的 Bark App，复制专属链接中最后那段随机字符串（例如链接是 `https://api.day.app/abcd1234efgh/`，那么只填写 `abcd1234efgh` 进去）。
5. 点击 **Add secret** 保存。

### 第 5 步：安装至手机桌面

1. 在您的 iPhone (建议使用 Safari) 或 Android (建议使用 Chrome/Edge) 上打开刚才第三步生成的 **GitHub Pages 链接**。
2. 点击浏览器底部的**分享**图标（iPhone）或右上角的**菜单**（Android）。
3. 选择 **“添加到主屏幕” (Add to Home Screen)**。
4. 返回桌面，您就可以像打开一个普通 App 一样每天快速查看当天的美股定投评估结果了！

## 👨‍💻 计算模型详述 (Model 4.3.1 非对称极值搜集)

本版本摒弃了早期粗暴的“过热即空仓”逻辑，采用了**“防拖累底仓 + 极端左侧狂暴抄底”**的非对称数学模型。这意味着即使面对长达两年的单边疯牛，策略也绝对不会踏空；而在遭遇史诗级黑天鹅时，买入倍数可飙升至极值。

### 核心计算公式一览

不同指数对下列三个因子的得分进行加权时，采用的**风格倾斜权重系数 (Style-Tilted Weights)**：

| 指数                | 估值权重 ($W_{PE}$) | 情绪权重 ($W_{VIX}$) | 趋势权重 ($W_{Bias}$) | 风格特征                                         |
| :------------------ | :-----------------: | :------------------: | :-------------------: | :----------------------------------------------- |
| **纳斯达克 (NDX)**  |         30%         |         30%          |        **40%**        | **趋势优先**：顺应科技股高动量与长牛惯性         |
| **标普500 (SP500)** |       **40%**       |         30%          |          30%          | **价值优先**：顺应白马股的极其可靠的均值回归特性 |

---

### 因子 1: 估值因子 (Valuation) 
- **数据源**：蛋卷基金接口获取的「PE 历史百分位」(数值越大越便宜)。
- **防拖累逻辑**：极度便宜时加注（最高 2.0x），即便极度昂贵泡沫，也**永远保留 0.5x 防现金拖累底仓**，确保吃透牛市。
  - **极度便宜** ($x > 70\%$): `Score = 1.0 + ((x - 0.7) / 0.3) * 1.0`
  - **价值合理** ($30\% <= x <= 70\%$): `Score = 1.0`
  - **极度高估** ($x < 30\%$): `Score = 0.5 + (x / 0.3) * 0.5`

### 因子 2: 市场情绪因子 (Sentiment)
- **数据源**：Yahoo Finance 的波动率指数 (VIX/VXN)。
- **狂暴抄底逻辑**：专门应对由突发利空引发的断崖暴跌踩踏，进行脉冲式抄底，最高爆发至 **2.5x**。
  - **极度惊恐** ($x > 30$): `Score = min(2.5, 1.8 + (x - 30) / 10.0)`
  - **恐惧担忧** ($20 < x <= 30$): `Score = 1.0 + ((x - 20) / 10) * 0.8`
  - **正常波动** ($14 <= x <= 20$): `Score = 1.0`
  - **极度贪婪** ($x < 14$): `Score = 0.8`

### 因子 3: 趋势因子 (Trend/Bias)
- **数据源**：价格偏离 200日均线 的乖离率。
- **顺势回撤逻辑**：专门应对“钝刀子割肉”的漫长阴跌，或者极度疯涨的单边牛市。
  - **黄金坑底** (跌破 10% 以上): `Score = 2.0`
  - **破位超跌** (跌破 0% ~ 10%): `Score = 1.0 + (|x| / 0.10) * 1.0`
  - **顺势而为** (均线之上 0% ~ 10%): `Score = 1.0`
  - **过热回撤** (均线之上 10% ~ 20%): `Score = 1.0 - ((x - 0.1) / 0.1) * 0.5`
  - **极度泡沫** (偏离均线 20% 以上): `Score = 0.5`

### 最终判决：综合资金乘数
- 将这三个维度的得分系数，分别与对应指数的独立加权占比相乘：
$$ 最终建议投入倍数 = \frac{S_{PE} \cdot W_{PE} + S_{VIX} \cdot W_{VIX} + S_{Bias} \cdot W_{Bias}}{W_{PE} + W_{VIX} + W_{Bias}} $$

计算结果会严格圈定在 `0.0x` 到 `3.0x` 安全倍率范围内。您只需将每日邮件/网页上的倍率，乘以您的基础定投金额，即为当天的最佳加仓策略！
